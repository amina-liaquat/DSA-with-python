# DSA With Python

**Data Structures** are ways of storing and organizing data in a computer so that it can be accessed and modified efficiently.

In Python, there is built-in support for several data structures, including:

- Lists
- Dictionaries
- Sets
- Tuples

In addition, more complex data structures can be implemented using classes and objects, such as:

- Linked Lists
- Stacks
- Queues
- Trees
- Graphs
